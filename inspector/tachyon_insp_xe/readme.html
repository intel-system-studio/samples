Intel® Inspector Sample

Tachyon is a parallel raytracer / renderer used in combination with the Intel® Inspector tutorial.

This example includes software developed by John E. Stone. See here for copyright information.

Quick Start on Windows*
  1. Install Intel® Parallel Studio XE
  2. Copy the file tachhyon_samples.zip and extract the contents of the file to a folder where you have write permissions.
  3. Start Visual Studio and open the solution file tachyon_samples\tachyon_samples.sln.
  4. Use the project as directed in the Intel Inspector XE tutorial.
Quick Start on Linux*
  1. Install Intel® Parallel Studio XE
  2. Copy the file tachyon_samples.tar.gz and extract the contents of the file to a folder where you have write permissions.
  3. Use the project as directed in the Intel Inspector tutorial.
