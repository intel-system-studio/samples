Matrix Multiply

This sample demonstrates the use of Intel® Cilk™ Plus keyword "cilk_for" to multiply two matrices in parallel.

System Requirements:
  • Hardware:
    • Any Intel® processor like 2nd Generation Intel® Core™ i3, i5, or i7 processors and Intel® Xeon® E3 or E5 processor family
  • For Microsoft* Windows*:
    • Microsoft Visual Studio 2012*, 2013* or 2015* Professional Edition or above
    • Intel® Parallel Studio XE 2017 Composer Edition for C++ Windows* or newer
      or
      Intel® System Studio 2017 Composer Edition for Windows* or newer
  • For Linux*:
    • GNU* GCC 4.5 or newer
    • Intel® Parallel Studio XE 2017 Composer Edition for C++ Linux* or newer
      or
      Intel® System Studio 2017 Composer Edition for Linux* or newer
  • For OS X*:
    • OS X 10.9 or above
    • Xcode* 5.0 or above
    • Intel® Parallel Studio XE 2017 Composer Edition for C++ OS X or newer

Build Instructions

For Microsoft* Visual Studio users:
  • Open Visual Studio and load the solution .sln file
  • Choose a configuration (for best performance, choose a release configuration):
    • Debug and Release: uses Intel C++ compiler
  • Run application: press Ctrl+F5 to run the application.
For Windows Command Line users:
  • Open the appropriate Intel C++ compiler command prompt
  • Build and run the application:
    build
    build run
  • (Optionally) Build without Optimization and run the application:
    build debug
    build run
For Linux* or OS X* users:
  • From a terminal window, navigate to the project folder
  • Set the environment: source <icc-install-dir>/bin/compilervars.sh ia32 or intel64
  • Build and run the application:
    make
    make run
  • (Optionally) Build without optimization and run the application:
    make debug
    make run debug=1
