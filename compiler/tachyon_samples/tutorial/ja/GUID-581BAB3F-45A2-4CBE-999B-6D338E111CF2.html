シリアルバージョンをビルドする

シリアルバージョンをビルドするには、シリアルバージョンのサンプルをコンパイルします。サンプル・アプリケーションを実行して、アプリケーションの実行時間を記録します。この時間が、パフォーマンスのベースラインになります。

  1. build_serial プロジェクトをスタートアップ・プロジェクトに設定します ([プロジェクト] > [スタートアップ プロジェクトに設定])。

  2. 構成を Release モードにして ([ビルド] > [構成マネージャ] > [アクティブ ソリューション構成:] > Release)、build_serial プロジェクトをビルドします。

  3. [デバッグ] > [デバッグなしで開始] で build_serial.exe を実行します。ウィンドウタイトルに表示された時間 (秒) を記録します。このイメージをレンダリングする時間が、Microsoft* Visual C++* コンパイラーで並列化する際のベースラインになります。

  4. プロジェクト build_serial と tachyon.common で使用するコンパイラーをインテル® C++ コンパイラーに変更します ([プロジェクト] > [Intel Compiler (インテル® コンパイラー)] > [Use Intel C++ (インテル® C++ を使用)])。

  5. Release モードで build_serial をリビルドします (ここでは、インテル® コンパイラーを使用)。

  6. サンプル・アプリケーションを開始します。

イメージのレンダリングにかかった時間を記録します。この時間が、パフォーマンスのベースラインになります。
