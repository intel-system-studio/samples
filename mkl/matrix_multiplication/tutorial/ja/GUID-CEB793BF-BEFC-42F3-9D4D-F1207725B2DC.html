Basic Linear Algebra Subprograms (BLAS)

Basic Linear Algebra Subprograms (BLAS) は、さまざまなベクトル演算と行列演算を実行する主要なライブラリーです。このチュートリアルでは、dgemm ルーチンを使用して行列演算を効率良く実行する方法を説明します。
