The tachyon sample is a parallel raytracer / renderer used in combination with the Intel® VTune Amplifier for Systems tutorial "Finding Hotspots on an Android* Platform" on analyzing a target Android application for problem areas from a Windows* or Linux* host system. The tutorial in this package shows how to set up a remote Android analysis, create a project on the host system, run a Basic Hotspots analysis, and interpret the results data step-by-step.

System Requirements

  • Host System:

    • Supported Intel® processor. Refer to the Release Notes available in your product installation files for more information
    • Eclipse* Mars.1 release (4.5.1) or later or Apache Ant* release 1.9.6 or later
    • Linux host: Intel® System Studio 2018 Professional or Ultimate Edition for Linux
    • Windows host: Intel® System Studio 2018 Professional or Ultimate Edition for Windows
  • Target System:
    • Android device with Intel processor
    • Android version 4 or later

Build Instructions

  • Edit local.properties to reflect the path where your NDK and SDK are installed.
  • Import the project in Eclipse

    • Note: Detailed build instructions are available in the tutorials listed below.

    Related Tutorial

    The tachyon sample for the VTune Amplifier is accompanied by the following tutorial:

    • Finding Hotspots on an Android Platform: Identify where your application is spending time, detect the most time-consuming program units and how they were called.
