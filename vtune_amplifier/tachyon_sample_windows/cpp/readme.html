Intel® VTune™ Amplifier Tachyon Sample

The tachyon sample is a 2-D raytracer/renderer that shows how different types of Intel® VTune™ Amplifier analysis help identify performance bottlenecks in the code, how algorithm optimization techniques affect the application performance, and how you can localize the performance changes you got after optimizations.

System Requirements

  • Hardware:

    • For the most up to date hardware requirements, see the Intel® VTune™ Amplifier release notes.

  • Software:

    • Microsoft* Visual Studio 2013 or higher

Build Instructions

From the parent folder of Tachyon sample, open the appropriate vc folder depending on the version of Microsoft* Visual Studio installed on your computer. For instance, if you have Visual Studio 2013 installed, open the vc12 folder.

Note:

Detailed build instructions are available in the tutorials listed below.

Related Tutorials

The tachyon sample for the VTune Amplifier is accompanied by the following tutorials:

  • Analyze Locks and Waits: Identify locks and waits preventing parallelization.

  • Finding Hotspots: Identify where your application is spending time, detect the most time-consuming program units and how they were called.
