Tutorial: Using Intel® System Studio Components for Matrix Multiplication Computation

Introduction

The Intel® System Studio is a comprehensive and integrated tool suite that provides developers with advanced system tools and technologies to help accelerate the delivery of the next generation power efficient, high performance, and reliable embedded and mobile devices. It allows you to build, optimize, debug, tune, and deploy 32-and 64-bit C and C++ system and application code on Windows* and Linux* host platforms for running on Linux*, Android*, and Windows* targets.

The Matrix Multiplication example supplied with the package illustrates how the Intel® System Studio components - Intel® Compiler for Systems, Intel® Math Kernel Library, and Intel® VTune™ Amplifier for Systems - can be used together for optimizing and analyzing the sample code.

The sample computes the matrix product using a triple nested loop.

Where:

ISS_01_compiler_mtarix_multiply (Folder) – Simple C++ implementation of computation of matrix product.

ISS_02_mkl_matrix_multiply (Folder) – a matrix multiplication sample code with optimized Intel® Math Kernel Library functions.

Terminology

This tutorial discusses Host and Target systems. The Host system is the Linux* or Windows* desktop computer that you use for compiling the sample. The Target system is an x86-based hardware, on which the sample will be running.
