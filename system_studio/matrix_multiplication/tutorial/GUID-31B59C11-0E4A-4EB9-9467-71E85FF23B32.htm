Tutorial: Using Intel® System Studio Components for Matrix Multiplication Computation

Building and Running the Sample from the Command Line on the Host

Open a terminal and run below script:

source <installdir>/bin/compilervars.sh (arch)

where (arch) is one of the following:

ia32 : setup for IA-32 architecture

intel64 : setup for Intel® 64 architecture

You must choose one of {ia32,intel64} depending on the target architecture to be used. Environment setup needs only be performed once per shell window to be used.

Use the following instructions to build the sample from command line:

ISS_01_compiler_matrix_multiply: 

>icpc -O2  -c  -o compiler_sample.o  compiler_sample.cpp
>icpc  -o  ISS_01_compiler_matrix_multiply  compiler_sample.o
>./ ISS_01_compiler_matrix_multiply

ISS_02_mkl_matrix_multiply 

>>icpc  -O2   -mkl=parallel  -c   -o mkl_sample.o  mkl_sample.cpp
>icpc  -mkl=parallel  -o  ISS_03_mkl_matrix_multiply  mkl_sample.o
>./ ISS_03_mkl_matrix_multiply
