Tutorial: Using Intel® System Studio Components for Matrix Multiplication Computation

Overview

Discover how various components of the Intel® System Studio can be used for optimizing and analyzing a code by running the Matrix Multiplication example.

About This Tutorial

This tutorial shows how a code can be optimized and analyzed using several components of the Intel® System Studio with the Matrix Multiplication example supplied in the package.

Estimated Duration

15-20 minutes

Learning Objectives

After you complete this tutorial, you will know how the matrix multiplication computation performance is improved by using:

  • The Intel® Compiler for Systems
  • The Intel® VTune™ Amplifier for Systems finding hotspots
  • The Intel® Math Kernel Library optimized functions

More Resources

The procedures in this tutorial apply to using the Matrix Multiplication sample. Refer to Intel® Software Product Samples and Tutorials https://software.intel.com/en-us/articles/intel-system-studio-tutorials to find samples and tutorials for the Intel® System Studio components.

In addition, you can find more resources at https://software.intel.com/en-us/intel-software-technical-documentation
